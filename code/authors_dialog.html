In order to use the Liveblog Add-on, you'll need to generate a google spreadsheet that follows the schema provided as an example here. The spreadsheet needs to be accesible to everyone that will work on the liveblog document so we recommend that you set the permissions to be viewable to everyone with the link. Basically it needs a header row and the following columns:

  • Initials (Required): Initials of the author 2-3 chars.
  • Name (Required): Name of the author.
  • Role (Optional): Role of the author.
  • Page (Optional): Page of the author to link to it.
  • Image (Optional): Image of the author.
